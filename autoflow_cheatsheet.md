# AutoFlow : The Ultimate Cheatsheet 🦾

Use these exact commands in the UI for the most stable and autonomous experience.

### 🎸 Music & YouTube
- **Command**: *"Play [Artist Name] lofi on YouTube"*
- **What happens**: Searches, clicks the first result, and **automatically skips ads**.
- **Tip**: You can follow up with *"Wait 60 seconds and then scrape the video comments"*.

### 📱 WhatsApp Web (Log-in Required)
- **Command**: *"Go to WhatsApp web and wait 15 seconds"*
- **What happens**: Opens your authenticated session.
- **Tip**: If it's your first time, scan the QR code within that 15-second window.

### 📁 File Mastering
- **Command**: *"Create a file on my desktop called 'notes.txt' with the text: Hello World"*
- **What happens**: Uses your OneDrive Desktop path automatically.
- **Command**: *"Find the absolute path of 'notes.txt'"*

### 💻 Developer Tools
- **Command**: *"Launch Claude Code"* 
- **What happens**: Starts the official `npx @anthropic-ai/claude-code` CLI.
- **Command**: *"Run command: git status"*

### 🧠 Troubleshooting
- **If the browser is closing too fast**: *"Go to [URL] and wait 10 seconds"*
- **If a click fails**: *"Search for [X], then scrape the page to see why it failed"*
